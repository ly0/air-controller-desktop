# 版本说明
air-controller 解决了 android 端到 macos 端浏览式传输的痛点，但是由于原项目作者似乎已经不再更新，我又迫切有这个需求，于是咔咔一顿改有了这个版本（0.5.0) 和对应的 air-controller-mobile.

[手机端](https://github.com/ly0/air-controller-mobile/)
